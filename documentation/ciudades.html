get /ciudades

Obtener la información de todas las ciudades existentes.

Es requerido tener el permiso para consultar ciudades.

Secured by oauth_2_0
  • Request
  • Response

Query Parameters

  • pagina: (number )

    Opcional. Numero de página a regresar.

  • tamanoPagina: (number- default: 20 )

    Opcional. Tamaño de la página a regresar. Solo es válido si se especifica el parámetro "pagina".

  • orden: (string- default: id )

    Opcional. Nombre del campo que será el criterio para realizar el paginado de los resultados. Sólo es válido si se especifica el parámetro "pagina"

  • filtro: (string )

    Filtrado de ciudades por nombre.

  • estado: (string )

    Código de tres letras (ISO 3166-2) del estado al que pertenece la ciudad. (ej. México)

  • pais: (string- default: MX )

    Código de dos letras (ISO 3166-1 alpha-2) del país al que pertenece la ciudad.

HTTP status code 200

Regresa una colección de ciudades.

Type: application/json
