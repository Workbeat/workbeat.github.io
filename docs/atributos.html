post /atributos

Crear un atributo.

Es requerido tener el permiso para crear atributos.

Se valida:

  • Que se especifique un nombre para el atributo.
  • Que el nombre del atributo no exceda de 80 caracteres.
  • Que no exista otro atributo con el mismo nombre.
  • Que se hayan indicado los usos que tendrá el atributo y sean valores permitidos.
Secured by oauth_2_0
  • Request
  • Response

Body

Type: application/json

Schema:

{ "$schema": "http://json-schema.org/schema",
  "type": "object",
  "description": "Atributo",
  "properties": {
    "nombre": { "type": "string", "maxLength": 80, "required": true },
    "usos": [
              { 
                "type": "object",
                "description": "Uso de un atributo",
                "properties": {
                  "nombre": { "type": "string", "enum": ["Posición", "Empleado", "Descripción de puesto"], "required": true }
                },
                "required": true
              }
            ],
    "copiarDePosicionReporta": { "type": "boolean" },
    "copiarDeDescripcionPuesto": { "type": "boolean" }
  }
}

Example:

{
"nombre": "Sucursal", 
"usos": [
          {
          "nombre": "Posición"
          },
          {
          "nombre": "Empleado"
          }
        ],
"copiarDePosicionReporta": false,
"copiarDeDescripcionPuesto": false
}
