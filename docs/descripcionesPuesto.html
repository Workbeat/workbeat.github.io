/descripcionesPuesto/{id} post delete get

post

Editar una descripción de puesto.

Validar que: -Que exista un perfil psicométrico con este ID. -No Longitud máxima es de 100 caracteres. -No debe existir otra descripción de puesto con el mismo nombre en la organización. -Que exista una organización con este ID. -Por cada atributo a asignar se debe especificar ya sea el ID o la referencia del elemento a asignar. -Nivel de estudios: Sólo se aceptan los valores “Primaria”, “Secundaria”, “Carrera técnica”, “Preparatoria”, “Profesional”, “Especialidad”, “Maestría”, “Doctorado” o “No especificado”. -Sólo se aceptan los valores “NE”, “SOL”, “CAS”, “VIU”, “DIV”, “UL” o “SEP”. -Sólo se aceptan los valores “Trunco”, “Pasante” o “Titulado”. -Sólo se aceptan los valores “NE”, “M” o “F”.

delete

Borrar una descripción de puesto.

Es requerido tener el permiso para borrar descripciones de puesto.

Se valida:

  • Que la descripción de puesto no tenga posiciones relacionadas.
  • Que la descripción de puesto no tenga empleados relacionados.
get

Obtener la información básica de una descripción de puesto.
