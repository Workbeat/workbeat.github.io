  }

HTTP status code 400

Valida si el valor del id de la razón social no es númerico, o el valor del id del empleado no es alfanumérico de máximo 20 caracteres.

Body

Type: application/json

Example:

{
  "error": {
    "code": 400,
    "message": "La petición no está formada correctamente. El id de la razón social debe ser un número entero. El id del empleado debe ser una cadena de máximo 20 caracteres”.     
    }
}

HTTP status code 404

Si no se encuentra el empleado, sea igual al valor de id de la razón social y cuyo campo “idEmpleado” sea igual al valor del id del empleado, muestra un mensaje de error.

Body

Type: application/json

Example:

{
  "error": {
    "code": 404,
    "message": "No se encuentra el empleado en nómina."
           }
}
