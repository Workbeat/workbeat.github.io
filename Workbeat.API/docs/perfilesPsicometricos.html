/perfilesPsicometricos/{id} get

get

Obtener la informacion básica de un Perfil Psicometrico.

get /perfilesPsicometricos/{id}

Obtener la informacion básica de un Perfil Psicometrico.

Secured by oauth_2_0
  • Request
  • Response

URI Parameters

  • id: required (string )

HTTP status code 200

Debera obtener la información un perfil psicometrico correctamente.

Body

Type: application/json

Schema:

[
  { "$schema": "http://json-schema.org/schema",
    "type": "object",
    "description": "Perfil Psicometrico",
    "properties": {
      "id": { "type": "number" },
      "nombre": { "type": "string" },
      "descripcion": { "type": "string" },
      "fechaUltimoCambio": { "type": "date" },
     
    }
  }
]

Example:

[
  {
  "id": 205,
  "nombre": "Analista",
  "descripcion": "Analista de diferentes áreas de una empresa",
  "fechaUltimoCambio": "Thu, 11 Jun 2015, 12:13:00 GMT",
 
  }

HTTP status code 400

Valida si el valor del id no es númerico.

Body

Type: application/json

Example:

{
  "error": {
    "code": 400,
    "message": "La petición no está formada correctamente. El id del perfil psicométrico debe ser un número entero.”.      
    }
}

HTTP status code 404

Si no se encuentra el perfil psicometrico, muestra un mensaje de error.

Body

Type: application/json

Example:

{
  "error": {
    "code": 404,
    "message": "No se encuentra el perfil psicometrico."
           }
}
